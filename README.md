# Project Form Input PHP

## Deskripsi
Project ini merupakan form input sederhana menggunakan PHP dengan konsep Object Oriented Programming (OOP).

## Fitur
- Input data user
- Proses menggunakan class PHP
- Menampilkan hasil setelah submit

## Teknologi
- PHP
- HTML
- CSS

## Cara Menjalankan
1. Jalankan di localhost (XAMPP/Laragon)
2. Buka folder project
3. Akses melalui browser
